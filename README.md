# ATS Resume Builder — ResumeGenius

**Live URL:** _Your Vercel URL after deployment_

A free, instant, ATS-optimized resume builder. No sign-up. No data stored. Works entirely in your browser.

## 🚀 Features

- **Multi-step form** — Personal Info → Work Experience → Education → Skills → Summary → Certifications
- **Live preview** — Real-time resume preview as you type
- **ATS-optimized template** — Clean, single-column layout that passes all major ATS parsers
- **PDF download** — One-click, properly formatted PDF export
- **ATS score meter** — Real-time score showing how ATS-ready your resume is
- **ATS tips checklist** — Expert guidance baked in
- **100% free** — No subscriptions, no sign-up, no data stored

## 🛠 Tech Stack

- Plain HTML + CSS + JavaScript (zero dependencies, zero build step)
- `html2pdf.js` via CDN for PDF export
- Deployed on Vercel (static)

## 📦 Deployment

### Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import the GitHub repo
4. Vercel auto-detects static site — click **Deploy**
5. Your site is live! ✅

### Local Development

```bash
# Any static file server works
npx serve .
# or just open index.html in your browser
```
